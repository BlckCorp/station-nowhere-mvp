# Станция Нигде / Station Nowhere

**Жанр:** browser roguelike / survival road game  
**Сеттинг:** постапокалиптическая Россия после Зимы Связи  
**Формат MVP:** статичная HTML/CSS/JS-игра для GitHub Pages

## Игровой крючок

Главная механика MVP — **память маршрута**.

После смерти игрок начинает новый забег почти с нуля, но сохраняет обрывки памяти: слухи, слабости врагов, отметки о тайниках и предупреждения о станциях. Память не всегда достоверна: часть подсказок может врать, потому что мир сам искажён рельсовой болезнью.

## Управление

- **WASD / стрелки** — движение
- **Пробел / Enter** — взаимодействовать с объектом на клетке
- **R** — отдохнуть на станции
- **M** — открыть / закрыть журнал памяти
- **N** — начать новый забег

## MVP-фичи

- процедурная карта перегонов и станций;
- пошаговое движение по сетке;
- 3 типа врагов с разным поведением;
- ресурсы: здоровье, топливо, патроны, усталость;
- события на клетках: станции, тайники, радио, дома, блокпосты, аномалии;
- смерть и новый забег;
- сохранение фрагментов памяти в `localStorage`;
- лор-журнал и цель добраться до Станции Нигде.

## Как запустить локально

Просто открой `index.html` в браузере.

## Как включить GitHub Pages

1. Открой настройки репозитория.
2. Перейди в **Settings → Pages**.
3. В **Build and deployment** выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Сохрани.

После публикации игра будет доступна примерно по адресу:

```text
https://blckcorp.github.io/station-nowhere-mvp/
```

## Файлы проекта

- `index.html` — интерфейс игры;
- `styles.css` — визуальный стиль;
- `game.js` — игровая логика;
- `LORE.md` — расширенный лор мира;
- `DESIGN.md` — геймдизайн-документ MVP.
