# Станция Здесь / Station Here

**Подзаголовок:** дорога к Нигде / the road to Nowhere  
**Жанр:** 2D top-down life sim / мрачный Stardew-like / post-rail colony sim  
**Сеттинг:** постапокалиптическая Россия после Зимы Связи  
**Формат MVP:** статичная HTML/CSS/JS-игра для GitHub Pages

## Главная идея

Игрок не просто едет к Станции Нигде. Он получает заброшенную станцию на мёртвой ветке и пытается сделать из неё живое место.

**Здесь** — то, что ты строишь.  
**Нигде** — то, что зовёт тебя уехать.

Днём игра почти уютная: грядки, ремонт, жители, вода, еда, мастерская, дрезина. Ночью возвращается хоррор: радио говорит голосами умерших, Путевой Смотритель проверяет рельсы, Домовой меняет комнаты, Молчащие Срочники требуют документы на станцию.

## Визуальная цель

Это не копия Stardew Valley, а духовный преемник в своём сеттинге:

- 2D top-down камера;
- тайловая карта;
- персонаж ходит по станции;
- камера показывает только участок мира вокруг героя;
- постройки выглядят как игровые объекты, а не как пункты меню;
- грядки, рельсы, лес, снег, водокачка, депо и радио имеют собственные CSS-пиксельные тайлы;
- всё сделано своими средствами без копирования чужих ассетов.

## Игровой цикл MVP

Каждый день делится на четыре фазы:

1. **Утро** — проверка станции.
2. **День** — работа: посадки, ремонт, сбор ресурсов.
3. **Вечер** — разговоры с жителями и подготовка.
4. **Ночь** — расход еды/воды, рост грядок, аномальные события.

Каждое действие двигает время.

## Управление

### Клавиатура

- **WASD / стрелки** — движение по станции;
- **Space / Enter** — использовать выбранный инструмент;
- **1–6** — выбрать инструмент;
- **I** — инвентарь;
- **C** — крафт;
- **Q** — задания;
- **E** — маршруты дрезины;
- **R** — отдых у печки;
- **M** — память станции;
- **N** — новая станция / сброс сохранения;
- **Esc** — закрыть окно.

### Телефон

Внизу экрана есть D-pad и быстрые кнопки:

- движение;
- действие;
- отдых;
- память;
- новая станция.

## Как запустить локально

Просто открой `index.html` в браузере.

## Как включить GitHub Pages

1. Открой настройки репозитория.
2. Перейди в **Settings → Pages**.
3. В **Build and deployment** выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Сохрани.

После публикации игра будет доступна примерно по адресу:

```text
https://blckcorp.github.io/station-nowhere-mvp/
```

Чтобы сбросить кэш, открывай:

```text
https://blckcorp.github.io/station-nowhere-mvp/?v=7
```

## Файлы проекта

- `index.html` — интерфейс игры;
- `styles.css` — базовый визуальный стиль, CSS-пиксельные тайлы и 2D-сцена;
- `gamefeel.css` — toolbelt, модальные окна, погода, анимации, game-feel;
- `mobile.css` — сенсорное управление;
- `game.js` — игровая логика Station Here;
- `mobile.js` — обработка экранных кнопок;
- `LORE.md` — лор мира;
- `DESIGN.md` — геймдизайн-документ MVP.
